# 🎸 簡易ギターチューナー ＆ メトロノーム

<img src="https://livedoor.blogimg.jp/hiroburo3-netadouga/imgs/2/d/2d3330be.png?v=20260620144450" width="100%" alt="簡易ギターチューナー＆メトロノーム アイキャッチ">

ギターのレギュラーチューニング（開放弦）に対応した簡易チューナーと、テンポを自由に変更できるシンプルなメトロノームが一体になった便利なWebツールです。

## 🌟 主な機能
- **ギター用開放弦チューナー:** 6弦（E）から1弦（E）までの基準音（三角波）をワンタップで再生。
- **簡易メトロノーム:** 40 BPM 〜 240 BPM までスライダーで直感的に速度調整が可能。視覚的なインジケーター付き。
